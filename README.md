# Object-Detection-Alarm
A simple Arduino project to detect objects and sound the alarm using arduino UNO, Buzzer and IR infrared obstacle avoidance sensor

# Components
* Arduino UNO
* Jumper cable [Male and Female] x12
* LED x1
* Buzzer x1
* IR sensor x1
